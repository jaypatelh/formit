Final Prototype

Current Features

Dancer List

Every time a dancer is added to the list, a corresponding circle is added to the canvas.

Formation Canvas

Circles are labelled with the dancer's name and can be assigned a color to signify certain blocking. Canvas with a lock-to grid and bolded center lines so that formations can be easily made with precision.

Multi-select

Multi-select allows blocking to be assigned quicker. Circles are automatically de-selected once a color is assigned to that group. Clicking on a blank area of the canvas also de-selects.

Formation Capture

The current state of the formation is continuously being saved so that the user is free to play around with the formation. A formation is permanently saved by clicking the "+" button.

Labeling for blocking

"Blocking" is represented by different pre-set colors. Blocking is assigned by selecting one or multiple dancers on the canvas, and then selecting a color.

Dance Timeline

Timeline at the bottom of the screen to give user a chronological sense of where they are in the set as well as an easier way to navigate through formations. When passing through formations on the canvas, the circles animate the transitions to demonstrate the distance traveled by each dancer.

Finishing Touches

  • "Saved" navigational breadcrumb that fades in and out every time a new formation is added
  • Swap spots feature that's triggered by a rotation gesture to permanently switch two dancers' spots for the remainder of the dance
  • Compact and expand formation feature that's activated by zoom in/out gesture
  • Move a selected group of dancers all at once
  • Change "delete selected dancers" function from deleting dancers from canvas only to removing dancer from both canvas and list
  • Description box where choreographer can record notes for each formation
